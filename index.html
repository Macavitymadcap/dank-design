    Animations & Effects

    Card Hover Effect

    Cards have a shadow effect that changes on hover:

    .card {
    box-shadow: 0 4px 8px 0 var(--shadow);
    transition: 0.3s;
}

.card:hover {
    box-shadow: var(--shadow);
}

    Button Hover Effect

    Buttons have opacity and transform effects on hover:

    .btn:hover {
    opacity: 0.8;
    transform: translateY(-2px);
}
