Accessibility

Skip Navigation

A skip navigation link allows keyboard users to bypass repetitive navigation elements: 

<!-- Add at the beginning of the body -->
<a href="#main-content" class="skip-to-content">Skip to content</a>

<!-- Add id to your main content -->
<main id="main-content">
    ...
</main>

Try it yourself: Press Tab on page load to see the skip link appear.

Screen Reader Only Text

For visually hidden content that should be read by screen readers:

The button above has visually hidden text "Download" for screen readers, while showing only an icon to sighted users. 

/* CSS for screen reader only content */
.sr-only {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    white-space: nowrap;
    border-width: 0;
}

/* Usage example */
<button class="btn btn-primary">
    <span class="sr-only">Download</span>
    <svg aria-hidden="true">...</svg>
</button>

ARIA Live Regions

ARIA live regions announce dynamic content changes to screen readers:

This content will change when you press the button. 
<div 
    class="card" 
    x-data="{ 
        updateLiveRegion: function () { 
            htmx.find('#live-region-example')
                .innerHTML = 
                    'Content updated at ' + new Date().toLocaleTimeString();
        } 
    }"
>
