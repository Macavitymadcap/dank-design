Responsive Grid

The grid system adapts to different screen sizes using the following classes:

  • .col-* - For all screen sizes
  • .col-md-* - For medium screens and up (≥768px)
  • .col-sm-* - For small screens (≤576px)

Wrapped Row (Flex Layout)

The .wrapped-row class uses flexbox to create a responsive, evenly spaced row of cards or swatches. It automatically wraps items and ensures consistent spacing.
